Bun + Infisical Docker Image

Fast JavaScript runtime meets secure secrets management

Production-ready Docker image combining Bun 1.3.10 with Infisical CLI 0.43.58 for blazing-fast JavaScript execution and seamless secrets injection. Built on Alpine Linux for minimal size and maximum security.

Get Started Docker Hub
Bun 1.3.10 Infisical 0.43.58 Alpine Linux Multi-Arch

Quick Start

Pull the Image

docker pull sswork/bun-infisical:latest

Run with Infisical

docker run -e INFISICAL_TOKEN=your-token \
  -e INFISICAL_API_URL=https://app.infisical.com \
  sswork/bun-infisical:latest \
  infisical run -- bun your-app.js

Docker Compose

version: '3.8'
services:
  app:
    image: sswork/bun-infisical:latest
    environment:
      - INFISICAL_TOKEN=${INFISICAL_TOKEN}
      - INFISICAL_API_URL=${INFISICAL_API_URL}
    command: infisical run -- bun start

Key Features

⚡

Lightning Fast

Bun 1.3.10 delivers exceptional JavaScript/TypeScript performance with a fast bundler, test runner, and package manager built-in.

🔐

Secure by Design

Infisical CLI 0.43.58 seamlessly injects secrets at runtime. Environment variables are never committed to your codebase.

📦

Minimal Footprint

Built on Alpine Linux for a compact image size. Multi-architecture support for AMD64 and ARM64 platforms.

🚀

Production Ready

Automated builds, security scanning, and semantic versioning. Perfect for CI/CD pipelines and container orchestration.
