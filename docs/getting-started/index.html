Getting Started

Quick setup guide for Bun + Infisical Docker image

Prerequisites

🐳

Docker Installed

You need Docker installed and running on your system. The image supports Docker Engine 20.10+ and Docker Desktop 4.0+.

Install Docker

Pull the Image

Download the latest version of the Bun + Infisical Docker image from Docker Hub.

Pull the latest image

docker pull sswork/bun-infisical:latest

Pull a specific version

docker pull sswork/bun-infisical:1.3.10-0.43.58

Verify Installation

Run these commands to verify that Bun and Infisical are installed correctly in the image.

Check Bun version

docker run -it sswork/bun-infisical:latest bun --version

Expected output: 1.3.10

Check Infisical version

docker run -it sswork/bun-infisical:latest infisical --version

Expected output: 0.43.58

Run Your First Container

Start an interactive container and test both Bun and Infisical.

Run interactive shell

docker run -it sswork/bun-infisical:latest sh

Run a simple Bun script

docker run sswork/bun-infisical:latest \
  bun -e "console.log('Hello from Bun!')"

Run with mounted volume

docker run -it -v $(pwd):/app -w /app sswork/bun-infisical:latest bun app.js

Mounts current directory to /app and sets it as working directory

Run with Infisical (requires token)

docker run -e INFISICAL_TOKEN=your-token \
  -v $(pwd):/app -w /app \
  sswork/bun-infisical:latest \
  infisical run -- bun your-app.js

Injects secrets from Infisical before running your application
